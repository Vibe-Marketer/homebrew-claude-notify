# homebrew-claude-notify

Homebrew tap for [claude-notify](https://github.com/vibe-marketer/claude-notify) -- native macOS notifications for Claude Code and OpenCode.

## Install

```bash
brew tap vibe-marketer/claude-notify
brew install claude-notify
```

Then run the one-time setup:

```bash
claude-notify-setup
```

## Update

```bash
brew upgrade claude-notify
```

No need to re-run setup after upgrading.
